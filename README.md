# 每日AI壁纸画廊

## 上传到免费静态托管

### 🥇 推荐：Gitee Pages（国内最快）
1. 打开 https://gitee.com ，注册/登录
2. 新建仓库 → 上传 gallery 文件夹里所有文件
3. 进入仓库 → 服务 → Gitee Pages → 启动
4. 获得链接如 `https://yourname.gitee.io/wallpaper-gallery`

### 🥈 备选：腾讯云COS（免费额度）
1. 打开 https://console.cloud.tencent.com/cos
2. 创建存储桶 → 开启静态网站
3. 上传 gallery 文件夹里所有文件
4. 获得访问域名

### 🥉 备选：GitHub Pages
1. 新建仓库 → 上传文件
2. Settings → Pages → 选择分支 → Save
3. 获得链接如 `https://yourname.github.io/wallpaper-gallery`

---

上传后打开 index.html 即可看到画廊。每天新生成的壁纸可以手动替换 01-current.png 并更新 index.html 中的壁纸数据。
